# Mock Electrical and Electronics Ltd — Corporate Website

> Kenya's Trusted Leader in Vehicle Safety & Tracking Solutions

## About
Corporate website for Mock Electrical and Electronics Ltd — an automotive electronics and telematics company based in Kenya providing NTSA approved vehicle speed limiters, GPS vehicle trackers, and vehicle telematics solutions.

## Tech Stack
- React 18 + Vite 5
- Tailwind CSS 3
- Framer Motion 11
- React Icons

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Customization
- Replace image placeholders in `public/images/`
- Update colors in `tailwind.config.js`
- Modify company details in individual components
